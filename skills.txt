# Goal

Analyze enterprise Java applications.

## Tasks

- Review pom.xml
- Identify deprecated APIs
- Detect Java version
- Recommend dependency upgrades
- Identify WebSphere migration issues
- Produce modernization report

## Output

- Executive Summary
- Risks
- Required Code Changes
- Migration Complexity
